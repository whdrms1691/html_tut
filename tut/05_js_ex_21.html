이벤트

  1. 이벤트의 종류

    - 마우스 이벤트 / 키보드 이벤트 / Element 이벤트 / 사용자 이벤트

  2. 이벤트 단계

    2-1. Capturing

    2-2. Targeting

    2-3. Bubbling

  3. 이벤트 함수

    - on() 함수

    - $(대상).on('이벤트종류',함수);

  4. 이벤트 이름

    - blur : 포커스를 잃었을 때

    - click : 마우스를 클릭했을 때

    - focusin / focusout : 포커스를 얻었을 떄 / 잃었을 때

    - keydown / keyup : 키보드를 눌렀을 때 / 떼었을 때

    - mousedown / mouseup : 마우스 버튼 눌렀을 때 / 떼었을 때

    - mouseenter / mouseleave : 마우스 커서가 닿았을 때 / 떨어졌을 때 (bubbling 발생하지 않음)

    - mouseover / mouseout : 마우스 커서가 닿았을 때 / 떨어졌을 때 (bubbling 발생함 => 사용하지 않는것이 바람직)

    - resize : 크기가 변경될 때

    - scroll : 스크롤 될 때 ( 웹 페이지 전체 스크롤 - window 객체 )

    - load / unload : 대상이 로딩되었을 때 / 제거 되었을 때 ( 이미지가 모두 로딩된 이후에 코드가 실행되게 할 때 주로 사용 )

    - select : select 박스에서 목록 중 하나 선택했을 때
