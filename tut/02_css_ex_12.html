relative와 absolute 속성값의 기본 성격과 차이점
1. relative는 좌표로 이동배치 되어도 원래 자리의 영역을 유지 / absolute는 배치이동되면 원래영역을 유지하지 않음
2. relative는 parent를 기준으로 배치되지만, absolute는 body를 기준으로 배치됨
3. childe box에 position:absolute를 적용할 때 parent box도 position:relative/absolute 속성을 적용하면 childe box가 parent를 기준으로 배치됨
4. position 속성이 적용된 box와 적용되지 않은 box가 겹칠 때 position 속성이 적용된 box가 앞으로 나옴
5. position 속성이 모두 적용된 box가 겹칠때 HTML 순서상 아래 box가 앞으로 나옴
